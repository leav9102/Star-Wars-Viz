# Overview
R project for a visualization showing that the favorability of Darth Vader is much higher for people who viewed Star Wars Episode III: Revenge of the Sith. In this repository contains a "FinalData" quarto document and knitted pdf and a "FinalProject" quarto document and knitted pdf. The "FinalData" file implements R data cleaning and preparation methods to prepare the large Star Wars csv containing survey data for use. The "FinalProject" file loads the cleaned data and produces a detailed and thoughtful visualization for the survey results. The pdfs included show the result of running these files.

# Project Guidelines
This project aims to create a data communication on an interesting or fun topic.  

# Setup and Use
1. Download FinalData.qmd, FinalProject(2).qmd, and week7_starwars(1).csv
2. Import the downloaded files into an RStudio environment
3. Run FinalData.qmd
4. Run FinalProject(2).qmd
